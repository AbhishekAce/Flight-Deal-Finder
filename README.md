# ✈️ Flight Dealer Finder

**Flight Dealer Finder** is a tool that helps users discover the best flight deals from multiple sources and airlines. Whether you're a frequent traveler or planning a vacation, this project helps you find the cheapest and most convenient flights quickly and easily.

## 🌟 Features

- 🔍 Search for flights by destination, date, or flexible time range
- 💸 Compare prices across different airlines and booking platforms
- 🛫 Track price drops and receive alerts
- 📊 Visualize flight trends with interactive charts
- 📱 Responsive UI for mobile and desktop
- 🧠 Smart filters for stops, times, and airlines

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (if front-end is in React)
- [Python 3.9+](https://www.python.org/) (if using Python backend)
- Optional: API keys for flight data (e.g. Skyscanner, Amadeus, Kiwi)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/flight-dealer-finder.git
cd flight-dealer-finder

# Backend setup (Python)
cd backend
pip install -r requirements.txt

# Frontend setup (React)
cd ../frontend
npm install
